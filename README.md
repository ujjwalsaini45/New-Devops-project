🚀 DevOps Project | CI/CD on AWS
📌 Overview

Designed and implemented a CI/CD pipeline using AWS EC2, Jenkins, and GitHub to automate build and deployment workflows.
The pipeline is triggered on every code commit, ensuring fast, reliable, and consistent application delivery.

⚙️ Tech Stack
Cloud: AWS (EC2)
CI/CD: Jenkins
Tools: Git, GitHub, VS Code
OS: Ubuntu (Linux)
Scripting: Shell Scripting
💡 Key Highlights
Automated end-to-end build and deployment process
Integrated GitHub with Jenkins for continuous integration
Reduced manual intervention and deployment errors
Improved deployment speed and system reliability
🎯 Objective
To gain hands-on experience in DevOps practices, focusing on automation, continuous integration, and cloud-based deployment solutions.

📣 Career Objective

Actively seeking opportunities in DevOps and Software Development to contribute and grow in a dynamic environment.

🔖 Skills Demonstrated

DevOps • CI/CD • AWS • Jenkins • Linux • Git • Automation
